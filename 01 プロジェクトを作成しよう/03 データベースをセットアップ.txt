今は、Django がデフォルトで使う sqlite3 というデータベースを使うことにします。

この設定はすでに mysite/settings.py ファイルの中に次のように書かれています。

mysite/settings.py
DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.sqlite3',
        'NAME': BASE_DIR / 'db.sqlite3',
    }
}

ブログのデータベースを作成するには、コンソールで次のコードを実行してみましょう： python manage.py migrate (manage.pyファイルのあるdjangogirlsディレクトリにいる必要があります) 。 うまくいったら次のように表示されるでしょう：

command-line
(myvenv) ~/djangogirls$ python manage.py migrate
